About TKI Social API

Purpose

TKI Social API is an internal microservice designed to simplify social media integration for Tonka International applications. It provides a unified interface for posting content, fetching posts, and managing interactions across multiple social media platforms.

Architecture

The service uses a modular adapter pattern to isolate platform-specific implementations:

  • Instagram Adapter: Manages Instagram posts and stories
  • LinkedIn Adapter: Manages professional networking posts and engagement
  • Meta Adapter: Handles Facebook and Instagram via Graph API
  • Reddit Adapter: Enables community-based content sharing
  • Threads Adapter: Interfaces with Meta's Threads platform
  • TikTok Adapter: Supports both personal and business TikTok accounts
  • X Adapter: Interfaces with Twitter/X for microblogging
  • YouTube Adapter: Manages video posts and community engagement

Security

This API is for internal use only and requires authentication via the x-internal-secret header. All requests are rate-limited and logged for security and monitoring purposes.

Technology Stack

  • Runtime: Node.js
  • Framework: Express.js
  • Logging: Pino
  • Security: Helmet, CORS
  • Testing: Vitest

Webhook Support

The service includes webhook handlers for real-time event processing from social media platforms. This enables automated responses to comments, mentions, and other platform interactions.

Integration

TKI Social API integrates with other Tonka International services including:

  • Binder Service: For Twilio messaging integration
  • Portal Service: For configuration and secrets management

Contact

For technical support, API access, or integration questions, please contact your system administrator or the development team.
